Engines

Five structural theme engines

Each engine swaps far more than accent colors — corner radius, border weight, shadows, backdrop blur, fonts, and icon stroke weight all change. Click a chip or use the switcher above.

Modular Workspace

Drag-and-drop panels powered by react-grid-layout. Resize terminals, localhost tracker, and explorer side by side.

Iris Command Bar

Zero-token local macros and contextual Git suggestions. Type naturally — Iris ranks push, pull, and shell helpers from live repo state.

xterm.js Terminal

High-fidelity terminal emulation with per-engine color palettes tuned to match each structural theme's surface.

Tauri + React

Native desktop performance with a modern React UI. Built with Vite, Zustand, and @tauri-apps for a fast, extensible shell.
